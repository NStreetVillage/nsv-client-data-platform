# Tableau Visualizations - Future Feature

This folder is intentionally a placeholder for Tableau planning and assets.

The current app stores client imports, review queue records, source details, and
program metrics. Tableau should be connected only after the reporting schema or
approved database views are stable.

Possible future work:

- Active clients by program.
- Cross-program participation.
- Data quality and duplicate-resolution dashboards.
- Occupancy and service utilization summaries.
