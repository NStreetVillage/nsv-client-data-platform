# NSV Data Platform Roadmap

## Phase 1 - Data Foundation and Demo

Goal: centralize client identity and prove data can be imported, matched, searched, and connected across programs.

Core features:

- NSV Client ID
- File upload
- CSV/Excel preview
- Column mapping
- Client matching
- Review queue
- Client search
- Program interaction view

Deadline:

- Demo prepared by Monday, June 29

## Phase 2 - Reporting and Tableau

Goal: build reporting layer after the data is organized.

Core features:

- Occupancy reporting
- SNAP/Medicaid report
- HMIS data quality reporting
- Compliance dashboard
- Tableau dashboards once licenses are available

## Phase 3 - Staff Workflow Tools

Goal: create internal staff-facing tools.

Core features:

- Case manager search portal
- Client profile page
- Service plan tracking
- Assessment tracking
- Case note tracking

## Phase 4 - AI and Advanced Analytics

Goal: add AI after clean data exists.

Core features:

- Natural-language questions
- Report summaries
- Trend detection
- Predictive analytics
- Advanced automation
