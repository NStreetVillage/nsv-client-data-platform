# Project Explanation in Plain English

Right now, NSV client information is spread across HMIS, HTH, eCW, JotForm, and spreadsheets.

The goal of this project is to create one central place where those exports can be uploaded, cleaned, matched to the right client, and used for reporting.

This project is not trying to replace HMIS, HTH, eCW, or JotForm right now.

Instead, it sits beside those systems and helps NSV answer questions like:

- How many women are we serving?
- Which clients are connected to more than one program?
- Who receives SNAP or Medicaid?
- Which compliance items are missing or coming due?
- Which reports can be automated?

The first demo focuses on importing data, searching for a client, and showing that the same client can be connected across multiple programs.
